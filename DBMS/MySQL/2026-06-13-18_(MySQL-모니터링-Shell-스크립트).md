# MySQL 모니터링 Shell 스크립트 (mymon)

- **카테고리**: #DBMS #MySQL #Monitoring
- **작성일**: 2026-06-13
- **참조 원본**: [[2026-06-12-mymon shell script for monitoring MySQL.md]]

## 1. 핵심 요약

mymon은 실시간 MySQL 모니터링 스크립트로, Threads, Questions, Slow queries 등을 5초 단위로 갱신 표시합니다.
Replication lag, 연결 수, 느린 쿼리 수 등 주요 지표를 한 화면에 표시합니다.

---

## 2. 주요 모니터링 항목

| 항목 | 설명 |
|------|------|
| Threads_connected | 현재 연결 수 |
| Questions | 누적 쿼리 수 |
| Slow_queries | Slow Query Log 카운트 |
| Innodb_buffer_pool_pages_dirty | Dirty pages (쓰기 대기) |
| Bytes_received/sent | 네트워크 I/O |
| Seconds_behind_master | Replication lag |

---

## 3. 기본 사용법

```bash
$ mymon --defaults-file=/etc/my.cnf -u root -p password -h localhost

# 5초마다 갱신되며 다음 정보 표시:
# - MySQL Version
# - Threads Connected
# - Questions/sec
# - Slow Queries
# - InnoDB Buffer Pool
# - Replication Status (if Slave)
```
