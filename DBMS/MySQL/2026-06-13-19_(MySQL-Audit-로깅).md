## 1. 핵심 요약

MySQL Audit Plugin은 쿼리 실행, 로그인, DDL/DML 명령을 기록합니다.
Percona, MariaDB, MySQL Enterprise 지원하며, 감사 이벤트를 XML 또는 JSON 형식으로 저장합니다.
GDPR/SOX 규정 준수 및 보안 감사에 필수입니다.

---

## 2. Percona Audit Plugin 설치

```sql
INSTALL PLUGIN audit_log SONAME 'audit_log.so';

-- 확인
SHOW PLUGINS;
```

---

## 3. Audit 설정

```sql
SET GLOBAL audit_log_events = 'CONNECT,QUERY,QUERY_DDL,QUERY_DML';
SET GLOBAL audit_log_format = 'XML';  -- or JSON
SET GLOBAL audit_log_file = '/var/log/mysql/audit.log';
SET GLOBAL audit_log_rotate_on_size = 1073741824;  -- 1GB
SET GLOBAL audit_log_rotations = 10;
```

---

## 4. 감사 이벤트 유형

| 이벤트 | 설명 |
|--------|------|
| CONNECT | 사용자 로그인/로그아웃 |
| QUERY | 모든 쿼리 |
| QUERY_DDL | CREATE/ALTER/DROP 등 |
| QUERY_DML | INSERT/UPDATE/DELETE 등 |
| QUERY_DCL | GRANT/REVOKE 등 |

---

## 5. Audit Log 분석

```bash
# XML 형식 로그 확인
cat /var/log/mysql/audit.log | grep -i "UPDATE\|DELETE"

# JSON 파싱
cat audit.log | jq '.audit_record[] | select(.command_class=="table_sql_statement")'
```
