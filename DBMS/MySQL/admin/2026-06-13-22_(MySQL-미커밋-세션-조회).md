## 1. 핵심 요약

COMMIT하지 않은 트랜잭션은 락을 유지하여 다른 쿼리를 블로킹합니다.
INFORMATION_SCHEMA.INNODB_TRX로 열려있는 트랜잭션 조회 및 강제 종료 가능합니다.

---

## 2. 미커밋 세션 조회

```sql
SELECT 
  trx_id, 
  trx_requested_lock_id, 
  trx_state,
  trx_started,
  TIME_TO_SEC(TIMEDIFF(NOW(), trx_started)) AS trx_sec,
  trx_mysql_thread_id,
  trx_query
FROM INFORMATION_SCHEMA.INNODB_TRX
ORDER BY trx_started;
```

---

## 3. 세션 강제 종료

```sql
KILL SESSION <mysql_thread_id>;
```
