# MySQL MY-011370·011355·010202 — keyring_file 초기화 오류

- **카테고리**: #DBMS #MySQL #troubleshooting
- **태그**: #MySQL #troubleshooting #error #keyring #plugin #startup
- **작성일**: 2026-06-14
- **참조 원본**: [[2026-06-14-MY-011370, MY-011355, MY-010202 keyring 오류 - BigDataTeam]]

## 1. 핵심 요약

MySQL 시작 시 `keyring_file` 플러그인이 활성화됐으나 keyring 파일 경로(`keyring_file_data`)가 없거나 권한이 없으면 `MY-011370·011355·010202` 에러 3종이 연속 발생한다. 해결은 디렉터리 생성 + 권한 부여 + `my.cnf` 설정 추가.

---

## 2. 에러 메시지

```
[ERROR] [MY-011370] Plugin keyring_file reported: 'File '/usr/local/mysql/mysql-keyring/keyring' not found (OS errno 2)'
[ERROR] [MY-011355] Plugin keyring_file reported: 'keyring_file initialization failure. Please check ...'
[ERROR] [MY-010202] Plugin 'keyring_file' init function returned error.
```

## 3. 조치

```bash
# 1) keyring 디렉터리 생성 + 권한 부여
mkdir -p /usr/local/mysql/mysql-keyring
chown -R mysql:mysql /usr/local/mysql/mysql-keyring
```

```ini
# 2) my.cnf 설정 추가
[mysqld]
plugin_dir=/usr/lib64/mysql/plugin
early-plugin-load=keyring_file.so
keyring_file_data=/usr/local/mysql/mysql-keyring/keyring
```
