## 1. 핵심 요약
Oracle 데이터 딕셔너리 뷰 레퍼런스. `USER_/ALL_/DBA_` 접두어로 범위를 구분하고, `V$` 뷰는 실시간 성능·상태 정보 제공. DB 관리(TBS·제어파일), 보안(사용자·권한·감사), 백업·복구(Undo·Redo·RMAN), 유틸리티(Data Pump·병렬처리) 4개 영역으로 구성. 마지막 섹션에 CDB/PDB 계층 구조와 주요 딕셔너리 뷰 관계를 나타낸 **Oracle 19c 주요 Dict ERD** 포함.

## 2. 데이터 딕셔너리 접두어 규칙

- **`USER_`**: 현재 로그인한 사용자가 **소유한** 객체 정보
- **`ALL_`**: 사용자가 소유했거나 **접근 권한을 가진** 모든 객체 정보
- **`DBA_`**: 데이터베이스 전체의 모든 객체 정보 (관리자 권한 필요)
- **`V$`**: 데이터베이스의 **실시간 성능 및 상태** 정보 (Dynamic Performance View)

## 3. DB 관리 및 구조 (Database & Storage)

### DB Creation (DB 생성 및 상태)
- `V$SGA`, `V$INSTANCE`: 인스턴스 정보 및 SGA 메모리 영역 모니터링
- `V$DATABASE`: DB 생성 정보, 모드(Log Mode), 컨트롤 파일 상태 확인
- `V$PROCESS`: 현재 활성화된 오라클 백그라운드/서버 프로세스 정보
- `V$SYSAUX_OCCUPANTS`: SYSAUX 테이블스페이스를 사용하는 구성 요소 확인

### TBS Management (테이블스페이스 관리) ✔️
- `DBA_TABLESPACES`, `DBA_TABLESPACE_GROUPS`: 테이블스페이스 및 그룹 정보 조회
- `DBA_DATA_FILES`, `V$DATAFILE`: 물리적 데이터 파일의 경로 및 상태 확인
- `DBA_FREE_SPACE`, `V$TABLESPACE`: 테이블스페이스별 남은 공간 모니터링
- `DATABASE_PROPERTIES`: DB 생성 시 설정된 영구적 속성(기본 시간대 등) 확인

### Storage Parameters (저장 공간 파라미터) ✔️
- `DBA_SEGMENTS`, `DBA_EXTENTS`: 테이블/인덱스 등이 차지하는 물리적 공간 및 세그먼트·익스텐트 정보
- `DBA_TABLES`, `DBA_INDEXES`: 테이블 및 인덱스의 물리적 구조 정보

### Control Files (제어 파일)
- `V$CONTROLFILE`, `V$CONTROLFILE_RECORD_SECTION`: DB 무결성 유지에 필수적인 컨트롤 파일의 경로 및 내부 레코드 섹션 정보

## 4. 보안·사용자·권한 관리 (Security & Privileges)

### Users & Resources (사용자 및 자원) ✔️
- `DBA_USERS`, `ALL_USERS`, `USER_USERS`: DB 사용자 계정 상태 관리
- `DBA_TS_QUOTAS`, `USER_TS_QUOTAS`: 사용자가 테이블스페이스를 사용할 수 있는 공간 할당량(Quota)
- `USER_PASSWORD_LIMITS`, `USER_RESOURCE_LIMITS`, `DBA_PROFILES`: 암호 정책 및 세션당 자원 제한 프로파일
- `RESOURCE_COST`: 자원 사용량에 따른 비용 가중치 정보
- `V$SESSION`, `V$SESSTAT`, `V$STATNAME`: 현재 접속 중인 세션 정보 및 세션별 통계 지표

### Roles & Privileges (역할 및 권한) ✔️
- `ALL_COL_PRIVS`, `USER_COL_PRIVS`, `DBA_COL_PRIVS`: 컬럼 단위로 부여된 권한 정보
- `ALL_TAB_PRIVS`, `USER_TAB_PRIVS`, `DBA_TAB_PRIVS`: 테이블/객체 단위로 부여된 권한 정보
- `DBA_ROLES`, `USER_ROLE_PRIVS`, `DBA_ROLE_PRIVS`: 정의된 롤(Role) 및 사용자에게 부여된 롤 정보
- `USER_SYS_PRIVS`, `DBA_SYS_PRIVS`: 시스템 권한(CREATE TABLE 등) 관리
- `SESSION_PRIVS`, `SESSION_ROLES`: 현재 로그인한 세션에 활성화된 권한과 롤 확인

### Auditing (감사)
- `STMT_AUDIT_OPTION_MAP`, `AUDIT_ACTIONS`: 감사(Audit) 옵션 및 작업 매핑 정보
- `DBA_STMT_AUDIT_OPTS`, `USER_OBJ_AUDIT_OPTS` 등: 시스템 및 객체에 설정된 감사 옵션 조회
- `USER_AUDIT_TRAIL`, `DBA_AUDIT_TRAIL` 등: 실제 기록된 감사 로그 데이터 확인

### Security (보안 기본)
- `DBA_USERS_WITH_DEFPWD`: 기본 패스워드(Default Password)를 그대로 사용하여 보안 위험이 있는 계정 조회

## 5. 백업·복구·트랜잭션 관리 (Backup & Transactions)

### Rollback Segments & Undo Management (롤백 및 언두 관리)
- `DBA_ROLLBACK_SEGS`, `V$ROLLSTAT`, `V$ROLLNAME`: 전통적인 롤백 세그먼트 상태 모니터링
- `DBA_UNDO_EXTENTS`, `V$UNDOSTAT`: 자동 언두(Undo) 공간 관리 및 데이터 보존 시간 모니터링
- `V$TRANSACTION`: 현재 실행 중인 활성 트랜잭션 정보

### Redo Log Files & Archived Redo Logs (리두 및 아카이브 로그)
- `V$LOG`, `V$LOGFILE`: 현재 사용 중인 온라인 리두 로그 그룹 및 파일 경로 상태
- `V$LOG_HISTORY`, `V$LOGHIST`: 리두 로그의 시퀀스 전환 이력
- `V$ARCHIVED_LOG`, `V$ARCHIVE_DEST`, `V$ARCHIVE_PROCESSES`: 아카이브 모드에서 저장된 로그 파일 및 목적지·프로세스 상태 확인

### RMAN Recovery Catalog (RMAN 백업 카탈로그)
- `RC_ARCHIVED_LOG`, `RC_BACKUP_SET`, `RC_DATAFILE` 등 (`RC_`로 시작하는 뷰): RMAN 복구 카탈로그 DB에서 백업된 파일·세트·컨트롤 파일 이력 조회

## 6. 유틸리티·동시성·튜닝 (Data Pump, Dispatchers & Tuning)

### Data PUMP (데이터 펌프 유틸리티)
- `DBA_DATAPUMP_JOBS`, `USER_DATAPUMP_JOBS`: 데이터 펌프(Export/Import) 작업의 진행 상태 모니터링
- `DBA_DIRECTORIES`: 데이터 펌프 등에서 OS 파일에 접근할 때 사용하는 오라클 디렉토리 객체 경로 확인

### Dispatchers (공유 서버 환경 디스패처)
- `V$DISPATCHER_CONFIG`, `V$MTS`, `V$DISPATCHER`: Shared Server(공유 서버) 환경에서 클라이언트 요청을 분배하는 디스패처 프로세스의 설정 및 상태 모니터링

### Tuning (병렬 처리 튜닝)
- `V$PX_PROCESS`, `V$PX_SESSION`: 병렬 쿼리(Parallel Execution)를 수행하는 슬레이브 프로세스와 세션의 실시간 상태
- `V$PX_PROCESS_SYSSTAT`: 시스템 전체의 병렬 처리 관련 통계 정보 분석
