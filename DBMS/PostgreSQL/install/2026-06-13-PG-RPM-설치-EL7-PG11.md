## 1. 핵심 요약

CentOS/RHEL 7(EL-7) 환경에서 PostgreSQL 11을 PGDG RPM 저장소로 설치하는 절차. postgres 유저 생성 → RPM 저장소 등록 → yum 설치 → initdb → 서비스 등록 → 원격 접속 설정 순서.

---

## 2. 설치 절차

```bash
# 1. postgres 유저 생성
groupadd -g 26 postgres
useradd -u 26 -g postgres -d /home/postgres postgres

# 2. PGDG 저장소 등록 (EL-7)
wget --no-check-certificate \
  https://download.postgresql.org/pub/repos/yum/reporpms/EL-7-x86_64/pgdg-redhat-repo-latest.noarch.rpm
sudo yum localinstall pgdg-redhat-repo-latest.noarch.rpm
# 또는: sudo rpm -Uvh https://download.postgresql.org/pub/repos/yum/.../pgdg-redhat-repo-latest.noarch.rpm

# 3. PostgreSQL 11 설치
sudo yum install -y postgresql11-server postgresql11-contrib

# 4. DB 초기화
sudo /usr/pgsql-11/bin/postgresql-11-setup initdb

# 5. 서비스 등록·시작
sudo systemctl start postgresql-11
sudo systemctl enable postgresql-11

# 6. postgres 비밀번호 변경
su - postgres -c 'psql'
# ALTER USER postgres PASSWORD '<new_password>';
```

## 3. 원격 접속 설정

```ini
# /var/lib/pgsql/11/data/postgresql.conf
listen_addresses = '*'

# /var/lib/pgsql/11/data/pg_hba.conf
# TYPE   DATABASE  USER  ADDRESS     METHOD
host     all       all   0.0.0.0/0   md5
```

```bash
# 방화벽 포트 개방
sudo firewall-cmd --permanent --zone=public --add-port=5432/tcp
sudo firewall-cmd --reload
```

## 4. postgres .bash_profile

```bash
PGDATA=/var/lib/pgsql/11/data
export PGDATA
PATH=$PATH:$HOME/.local/bin:$HOME/bin:/usr/pgsql-11/bin
export PATH
```
