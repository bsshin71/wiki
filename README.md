# Wiki for DBA
* 개인용으로 사용하는 private wiki 에서 공유할 만한 문서들을 선별해서  public repo 인 이 wiki 에  비정기적으로 sync 합니다.
* 개인 raw 문서자료를  LLM 이   핵심 내용을 요약한 형태로  가공한 문서라서 detail 한 내용이 빠진 부분이 많습니다. (요즘 hot 한 LLM wiki)
* 혼자서 여러가지 일을 하는 업무만 많이 하다 보니 깊이 있는 내용 보다는 다양한 주제 그리고 현업에서 자주 참고하는 자료 위주입니다.
* MySQL 도서, 웹 수집, 개인 작성 자료가 섞여 있습니다.
* MySQL,Postgresql,Airflow,ElasticSearch 등은  이전 직장에서 로그수집시스템 구축을 하면서 작성한 문서를 기반으로 만들어 졌기 때문에 유사한 시스템을 만들려는 분에게는 유용할 수 있습니다.
