## 1. 핵심 요약

Docker 데이터 영속성을 위한 3가지 볼륨 방식: ① **호스트 볼륨** (가장 간단, `-v 호스트경로:컨테이너경로`), ② **볼륨 컨테이너** (data-only 컨테이너, `--volumes-from`으로 공유), ③ **도커 볼륨** (`docker volume create`로 관리).

---

## 2. 호스트 볼륨 (가장 많이 사용)

```bash
docker run -d --name mydocker_db \
  -e MYSQL_ROOT_PASSWORD=password \
  -e MYSQL_DATABASE=wordpress \
  -v /home/wmydocker_db/:/var/lib/mysql \    # 데이터 디렉터리
  -v /root/mysql/conf/my.cnf:/etc/my.cnf \  # 설정 파일
  -p 4306:3306 \
  mysql:8.0.31

# my.cnf 추출 (첫 실행 후)
docker cp mydocker_db:/etc/my.cnf /root/mysql/conf/my.cnf
```

## 3. 볼륨 컨테이너 (--volumes-from)

```bash
# 볼륨 컨테이너 생성
docker run -d -it --name volume-container \
  -v /host/data:/app/data ubuntu

# 앱 컨테이너에서 볼륨 참조
docker run -d --name app --volumes-from volume-container my-app
```

## 4. 도커 볼륨 (docker volume)

```bash
docker volume create my-volume
docker volume ls
docker volume inspect my-volume

docker run -d --name app \
  -v my-volume:/app/data my-app
```
